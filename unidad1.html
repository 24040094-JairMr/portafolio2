Estándares SOA

XML

Lenguaje de marcado extensible para intercambio de datos

SOAP

Protocolo simple de acceso a objetos para comunicación

WSDL

Lenguaje de descripción de servicios web

UDDI

Descubrimiento e integración universal de servicios

REST

Transferencia de estado representacional para APIs
