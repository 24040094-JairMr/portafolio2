Propósito del Proyecto

Objetivos y alcance de nuestro portafolio académico

Objetivo General

Desarrollar una aplicación web que funcione como portafolio de evidencias académicas, presentando los contenidos de la materia "Desarrollo Web Orientado a Servicios" mediante buenas prácticas de diseño y desarrollo web.

Objetivos Específicos

  • Presentar de manera organizada las 4 unidades temáticas
  • Implementar un diseño web moderno y responsive
  • Demostrar el uso de tecnologías web actuales
  • Servir como evidencia de aprendizaje y competencias
  • Aplicar conceptos de arquitectura SOA y APIs REST
